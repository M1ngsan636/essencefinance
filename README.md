# Personal Finance Tracker

A simple ESSENCE application for tracking Cluster finances using SQLite database.

## Features
- Track income and expenses
- View transaction history
- Real-time balance calculation
- Category-based organization

## Deployment
This app is configured for deployment on Railway.app

## Local Development
1. Make sure you have PHP 8.0+ installed
2. Run: `php -S localhost:8000`
3. Visit: http://localhost:8000

## Environment Variables
- `PORT` - Server port (set by Railway)
- `RAILWAY_VOLUME_PATH` - Persistent storage path (set by Railway)
